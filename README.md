# Tennis Racket Finder

This project is now split into separate files so the quiz is easier to maintain.

## Files

- `index.html`: page structure
- `styles.css`: visual styling
- `script.js`: quiz logic, questions, and racket recommendations
- `tennis_racket_finder_v4.html`: original all-in-one version kept as a reference

## Run locally

Open `index.html` in your browser.
